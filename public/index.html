Workout Tracker API

This API powers a workout tracking app with user authentication, workout logging, routines, and metrics tracking.

Available Routes

  • User Module
    • POST /api/user/register - Register a new user
    • POST /api/user/login - Login and receive JWT
  • Workout Module
    • POST /api/workout - Create a workout session (can be partial, e.g. routineId only) (auth required)
    • PATCH /api/workout/:id - Incrementally update a workout session (add exercises, notes, metrics) (auth required)
    • GET /api/workout/:id - Get the current state of a workout session (auth required)
    • GET /api/workout/mine - Get all workouts for the authenticated user (auth required)
  • Routine Module
    • POST /api/routine - Create a routine (auth required)
    • GET /api/routine/:id - Get a routine by ID
    • GET /api/routine - List all routines
  • UserMetrics Module
    • POST /api/userMetrics - Create a metrics record (auth required)
    • GET /api/userMetrics/latest - Get latest metrics (auth required)
    • GET /api/userMetrics/history - Get metrics history (auth required)
    • PATCH /api/userMetrics/:id - Update metrics (auth required)

Functionalities

  • User registration and authentication (JWT)
  • Workout logging and history
  • Routine creation and reference
  • Metrics tracking (weight, BMI, body fat, personal bests)
  • Protected routes with auth middleware
  • Error handling for robust API responses
