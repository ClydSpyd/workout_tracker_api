Workout Tracker API

This API powers a workout tracking app with user authentication, workout logging, routines, and metrics tracking.

Available Routes

  • User Module
    • POST /api/user/register - Register a new user
    • POST /api/user/login - Login and receive JWT
    • GET /api/user/me - Get current user info
  • Workout Module
    • POST /api/workout - Create a workout session (can be partial, e.g. routineId only)
    • PATCH /api/workout/:id - Update a workout session (add exercises, notes, metrics)
    • POST /api/workout/:id/set - Add or update an exercise/set in a workout
    • GET /api/workout/:id - Get the current state of a workout session
    • GET /api/workout/mine - Get all workouts for the authenticated user
  • Routine Module
    • POST /api/routine - Create a routine
    • PATCH /api/routine/:id - Update a routine
    • GET /api/routine/:id - Get a routine by ID
    • GET /api/routine - List all routines
    • GET /api/routine/mine - List all routines created by the authenticated user
  • UserMetrics Module
    • POST /api/userMetrics - Create a metrics record
    • GET /api/userMetrics/latest - Get latest metrics
    • GET /api/userMetrics/history - Get metrics history
    • PATCH /api/userMetrics/:id - Update metrics

Functionalities

  • User registration and authentication (JWT)
  • Workout logging and history
  • Routine creation and reference
  • Metrics tracking (weight, BMI, body fat, personal bests)
  • Protected routes with auth middleware
  • Error handling for robust API responses

See API documentation for more details.
